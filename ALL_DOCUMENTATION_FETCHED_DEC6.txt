### 2. Cache Behavior Explained
**Critical finding:**
- Cache doesn't replicate outside originating data center
- Cache API not available in dashboard/playground
- Responses with Set-Cookie never cached
- Cache-Control headers control caching

**Why 308 redirects happened:**
- Old cache in edge servers
- Manual purge needed (no Cache API in Pages)
- Cache propagation takes time

---
