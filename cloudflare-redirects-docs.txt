<p>A placeholder can be defined with <code dir="auto">:placeholder_name</code>. A colon (<code dir="auto">:</code>) followed by a letter indicates the start of a placeholder and the placeholder name that follows must be composed of alphanumeric characters and underscores (<code dir="auto">:[A-Za-z]\w*</code>). Every named placeholder can only be referenced once. Placeholders match all characters apart from the delimiter, which when part of the host, is a period (<code dir="auto">.</code>) or a forward-slash (<code dir="auto">/</code>) and may only be a forward-slash (<code dir="auto">/</code>) when part of the path.</p>
<p>Similarly, the matched value can be used in the redirect values with <code dir="auto">:placeholder_name</code>.</p>
