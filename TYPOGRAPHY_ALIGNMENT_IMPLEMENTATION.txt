- Headings: #2563eb (blue)

- Links: #007acc (blue)
- White text on gradient: #ffffff

═══════════════════════════════════════════════════════════════════

HOW TO APPLY TO ALL PAGES:
===========================

Add these 2 lines to <head> of every page:

```html
<link rel="stylesheet" href="typography-standards.css">
<link rel="stylesheet" href="alignment-standards.css">
```

Order matters:
1. typography-standards.css (first - base styles)
2. alignment-standards.css (second - layout)
3. common-navigation.css (third - navigation)
4. common-footer.css (fourth - footer)
5. page-specific.css (last - overrides)

═══════════════════════════════════════════════════════════════════

AUTOMATED APPLICATION:
======================

Create script to add to all pages:

```javascript
// Add after <head> tag
const typographyLink = '<link rel="stylesheet" href="typography-standards.css">';
const alignmentLink = '<link rel="stylesheet" href="alignment-standards.css">';
```
