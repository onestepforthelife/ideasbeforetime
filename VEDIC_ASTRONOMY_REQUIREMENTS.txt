# VEDIC ASTRONOMY CALCULATOR - REQUIREMENTS

**Created:** December 5, 2025
**Purpose:** Educational astronomy tool with Vedic calendar integration
**Approach:** Science, mathematics, and cultural education

---

## MUST HAVE FEATURES:

### 1. Date/Time Input
- User enters: Date, Time, Place (city/coordinates)
- Can select ANY date (past, present, future)
- Timezone auto-detection
- Location search with autocomplete

### 2. 3D Solar System Visualization
- Shows planetary positions for selected date
- 9 planets + Moon + Sun
- Rahu & Ketu (lunar nodes) marked
- Interactive - rotate, zoom, click for info
- Real astronomical calculations

### 3. Vedic/Hindu Calendar Data
- **Tithi** (lunar day)
- **Nakshatra** (lunar mansion - 27 divisions)
- **Yoga** (combination of sun-moon)
- **Karana** (half of tithi)
- **Paksha** (bright/dark fortnight)
- **Masa** (month name)
- **Samvatsara** (year in 60-year cycle)

### 4. Regional Variations
- **North India** - Purnimanta system
- **South India** - Amanta system
- **West India** - Regional variations
- User selects region, calendar adjusts
