### Option B: Production Deploy (Cloudflare Workers)

**Step 1: Install Wrangler CLI**
```bash
npm install -g wrangler
```

**Step 2: Login to Cloudflare**
```bash
wrangler login
```

**Step 3: Navigate to Workers Folder**
```bash
cd workers
```

**Step 4: Set API Keys as Secrets**
