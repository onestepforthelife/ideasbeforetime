## üîß RUN & DEBUG CONFIGURATIONS

### Recommended Configurations to Add:

**1. Node.js: Run Current File**
```json
{
  "type": "node",
  "request": "launch",
  "name": "Run Current JS File",
  "program": "${file}",
  "cwd": "${workspaceFolder}/Cloudfare"
}
```

**2. Python: Run Current File**
```json
{
  "type": "python",

  "program": "${file}",

}
```

## üéØ HOW TO INSTALL

### Method 1: Extensions Tab
1. Click Extensions icon (Ctrl+Shift+X)
2. Search for extension name
3. Click "Install"
4. Reload Kiro if needed

### Method 2: Command Palette
1. Press Ctrl+Shift+P
2. Type: "Extensions: Install Extensions"
3. Search and install

### Method 3: Quick Install (All at Once)
Create a file `.vscode/extensions.json`:
```json
{
  "recommendations": [
    "ms-python.python",
    "dbaeumer.vscode-eslint",
    "esbenp.prettier-vscode",
    "ritwickdey.liveserver",
    "eamodio.gitlens",
    "github.vscode-pull-request-github",
    "humao.rest-client",
    "christian-kohler.path-intellisense",
    "wayou.vscode-todo-highlight",
    "msjsdiag.debugger-for-chrome"
  ]
}
```
