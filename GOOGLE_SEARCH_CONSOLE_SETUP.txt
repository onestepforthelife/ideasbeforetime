STEP 1: VERIFY OWNERSHIP
================================================================================

Option A: HTML File Upload (Recommended)
-----------------------------------------
1. Go to: https://search.google.com/search-console
2. Click "Add Property"
3. Enter: ideas-before-time.pages.dev
4. Choose "HTML file upload" method
5. Download the verification file (e.g., google1234567890abcdef.html)
6. Upload it to your Cloudfare folder
7. Deploy to GitHub
8. Click "Verify" in Search Console

Option B: Meta Tag (Alternative)
---------------------------------
1. Choose "HTML tag" method
2. Copy the meta tag provided
3. Add to <head> of index.html:
   <meta name="google-site-verification" content="YOUR_CODE_HERE" />
4. Deploy to GitHub
5. Click "Verify"

================================================================================
STEP 2: SUBMIT SITEMAP
================================================================================

After verification:
1. In Search Console, go to "Sitemaps" (left sidebar)
2. Enter: sitemap.xml
3. Click "Submit"
4. Wait 24-48 hours for Google to crawl

Your sitemap URL: https://ideas-before-time.pages.dev/sitemap.xml

================================================================================
STEP 3: REQUEST INDEXING (OPTIONAL - FASTER)
================================================================================

For immediate indexing of key pages:
1. Go to "URL Inspection" tool

================================================================================
TROUBLESHOOTING
================================================================================

If verification fails:
- Check file is in root directory
- Clear browser cache
- Wait 5 minutes and try again
- Ensure file is publicly accessible

If sitemap not processing:
- Check sitemap.xml is valid (use XML validator)
- Ensure robots.txt allows crawling
- Wait 24-48 hours
- Resubmit if needed
