## üí° TROUBLESHOOTING

**If PDF conversion fails:**
```bash
# Check if pdftotext installed:
pdftotext --version

# Or check if PyPDF2 installed:
python -c "import PyPDF2; print('OK')"

# If neither works, install one:
choco install poppler
# OR
pip install PyPDF2
```

**If MCP servers don't start:**
```bash
# Check Node.js version:
node --version
# Should be v18 or higher

# Restart Kiro IDE completely
```

**If memory doesn't persist:**
- Memory MCP stores in: `~/.mcp/memory/`
- Check if folder exists
- Restart Kiro if needed
