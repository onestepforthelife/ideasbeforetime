6. TROUBLESHOOTING
================================================================================

ISSUE: Navigation doesn't appear
SOLUTION:
- Check browser console (F12) for JavaScript errors
- Verify common-navigation.js uploaded correctly
- Clear browser cache (Ctrl+Shift+R)
- Check file paths are correct (no leading /)
- Wait 5 minutes for Cloudflare cache to clear

ISSUE: Meta tags don't appear in page source
SOLUTION:
- Verify page-metadata.js uploaded
- Check script is loaded before </body> tag
- View page source (Ctrl+U), not inspector
- Wait for Cloudflare cache to clear (5 min)
- Try incognito/private browsing mode

ISSUE: Social previews don't work
SOLUTION:
- Use Facebook Debugger to scrape again
- Check og:image URL is absolute (https://)
- Verify image file exists and is accessible
- Wait 24 hours for social media cache to clear
- Check image dimensions (1200x630 recommended)

ISSUE: Sitemap not processing in Search Console
SOLUTION:
- Verify sitemap.xml is valid XML
- Check all URLs are absolute (https://)
- Ensure robots.txt allows crawling
- Resubmit in Search Console
- Wait 24-48 hours for processing

ISSUE: Lighthouse SEO score not 100
SOLUTION:
- Check all meta tags present
- Verify meta descriptions 50-160 characters
- Ensure all images have alt text
- Check for broken links
- Verify mobile-friendly design

ISSUE: Images not lazy loading
SOLUTION:
- Check image-optimizer.js loaded
- Verify images have loading="lazy" attribute
- Check browser console for errors
- Test in Chrome (best lazy loading support)
