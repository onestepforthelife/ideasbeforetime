Vertical Navigation Demo

This shows how vertical tabs would look. Clean, modern, always visible.

Pros:

  • Always visible (no scrolling needed)
  • More space for navigation items
  • Modern, app-like feel
  • Clear active state
  • Easy to add more links

Cons:

  • Takes horizontal space
  • Different from current design
  • Needs mobile adaptation

Decision: Keep horizontal navigation (current) for consistency, or switch to vertical for modern look?
