PROBLEM: "I don't see Cloudflare Access login screen"
SOLUTION: 
- Wait 2-3 minutes for DNS to propagate
- Clear browser cache
- Try private/incognito window
- Make sure you deployed to Cloudflare Pages

PROBLEM: "Email not receiving OTP code"
SOLUTION:
- Check spam folder
- Make sure email is in allowed list
- Try different email

PROBLEM: "Access denied even with correct email"
SOLUTION:
- Check email is spelled correctly in policy
- Make sure policy action is "Allow" not "Block"
- Check application path is "/social-optimizer-*"
