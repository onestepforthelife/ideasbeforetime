ğŸ” WHAT IS CACHE PURGING?

Cache = Cloudflare saves copy of your pages
Problem = Old copies being served (redirects)
Solution = Delete all copies (purge)
Result = Cloudflare fetches fresh pages from GitHub

Think of it like:
- Cache = Old newspaper
- Purge = Throw away old newspaper
- Result = Get today's newspaper
