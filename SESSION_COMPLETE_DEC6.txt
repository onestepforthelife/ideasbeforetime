## ğŸ¯ KEY LEARNINGS

### Learning #39: _redirects File Can Cause 308 Redirects
**What happened:** `/*.html /:splat 200` rule caused redirects
**Why:** Cloudflare interpreted `:splat` as "remove .html extension"
**Fix:** Changed to `/* /:splat 200` to serve everything directly
**Prevention:** Test _redirects rules carefully before deploying

### Learning #40: Cache Can Persist After Deployment
**What happened:** Pushed fix but still seeing 308 redirects
**Why:** Cloudflare cache serving old version
**Fix:** Purge cache after every deployment
**Prevention:** Always purge cache after pushing changes
