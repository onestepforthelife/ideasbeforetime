üîß TROUBLESHOOTING
========================================

Problem: "git is not recognized"
Solution: Install Git or use GitHub Desktop's git
Edit bat file, change:
  git add .
To:
  "C:\Users\araag\AppData\Local\GitHubDesktop\app-3.3.6\resources\app\git\cmd\git.exe" add .

(Check your actual GitHub Desktop version number)

Problem: "Permission denied"
Solution: 
1. Open GitHub Desktop
2. Go to File ‚Üí Options ‚Üí Git
3. Make sure you're logged in
4. Try pushing manually once

Problem: Task doesn't run
Solution:
1. Check Task Scheduler ‚Üí Task History
2. Look for error messages
3. Make sure "Run whether user is logged on or not" is checked
4. Make sure you entered Windows password correctly
